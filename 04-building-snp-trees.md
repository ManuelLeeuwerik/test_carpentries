:::::::::::::::::::::::::::::::::::::: questions

- How does a SNP tree represent relatedness among *A. fumigatus isolates* ?

::::::::::::::::::::::::::::::::::::::::::::::::

::::::::::::::::::::::::::::::::::::: objectives

- Explain that a SNP tree is built from shared genome-wide variants. 
- Interpret tips, branches, clusters, and branch lengths. 
- Distinguish genetic relatedness from transmission direction. 
- Identify what epidemiological information is needed alongside the tree. 

::::::::::::::::::::::::::::::::::::::::::::::::

## Introduction
A SNP tree can show that isolates are closely related or distantly related, but it cannot by itself determine who infected whom. 

##  Practical
- Learners view a static tree first, then answer:
- Which isolates are nearest neighbours? 
- Which isolate is most distant from the suspected cluster? 
- Does the resistant isolate cluster with any susceptible isolate? 
- What metadata would help interpret this tree: date, environment, prior azole exposure, sampling source? 


::::::::::::::::::::::::::::::::::::: keypoints

- A SNP tree represents genetic relatedness. 
- Branch length reflects genetic difference, not necessarily calendar time. 
- Epidemiological interpretation requires metadata in addition to the tree.
