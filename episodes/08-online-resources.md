:::::::::::::::::::::::::::::::::::::: questions

- Which public resources can help interpret *A. fumigatus* genomic results?

::::::::::::::::::::::::::::::::::::::::::::::::

::::::::::::::::::::::::::::::::::::: objectives

- Identify databases relevant to genome sequences, gene function, typing, and resistance. 
- Recognise which question each resource can and cannot answer. 
- Use online resources to support, not replace, workflow-based interpretation. 

::::::::::::::::::::::::::::::::::::::::::::::::

## Introduction

##  Practical

Resource	Use
NCBI / GenBank	Reference genomes, assemblies, sequence records.
FungiDB	Fungus-specific genome resource, includes tools and a Galaxy-based workspace for analyses such as variant calling. 
PubMLST	MLST/typing context for *A. fumigatus*, with the discriminatory-power caveat. 
MARDy	Known fungal resistance mechanisms and antifungal drug associations. 
FungAMR	Curated fungal AMR mutation data, useful for looking up reported gene–drug–mutation associations. 
EUCAST / CLSI	Phenotypic antifungal susceptibility interpretation and breakpoints.


::::::::::::::::::::::::::::::::::::: keypoints

- Different databases answer different questions. 
- Resistance interpretation should be evidence-based and database-supported. 
- Online lookups should be documented so interpretations remain reproducible.
